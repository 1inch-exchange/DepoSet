# DepoSet
DepoSet is a dApp for ordering multi tokens in one single transaction.

# Sources
Logo: https://www.svgrepo.com/svg/193172/gas-mask

# Install
``yarn``

# Build
``yarn run build``

# Run dev
``yarn run start``
